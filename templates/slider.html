Usage

Wrap .b-slide components inside a .b-slider and the framework will do the rest.


<div class="b-slider">
    <div class="b-slide">
        # Content
    </div>
    <div class="b-slide">
        # Content
    </div>
    <div class="b-slide">
        # Content
    </div>
</div>

Under the Hood

We use Slick behind the scenes to make the slider work.

Go to Slick Docs


$('.b-slider').slick({
    arrows: true,
    draggable: false,
    swipeToSlide: true,
    autoplay: true,
    autoplaySpeed: 3000,
    responsive: [
        {
            breakpoint: 800,
            settings: {
                draggable: true,
            }
        }
    ]
});
